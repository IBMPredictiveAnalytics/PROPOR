Пропорциональный доверительный интервал

Эта процедура создает биномиальные и Пуассоновские доверительные интервалы и различия между значениями.

• Входные данные для этого диалогового окна не поступают из набора данных.

Количество числителей и количество знаменателей Данные для этого вычисления - числитель и знаменатель для пропорции. Если в каждом поле - одно число, генерируется таблица из одной строки. Если в каждом поле несколько чисел, в этой таблице будет по одной строке для каждого числа, и представляется сравнение каждого доверительного интервала с первым.

• Для сокращения ввода, если в поле введено только одно число, а в другом поле - несколько чисел, одно число будет повторено столько раз, сколько требуется.

Доверительный уровень Введите доверительный уровень (альфа), используемый для интервалов, в виде числа от 0 до 1.

Биномиальные доверительные интервалы используют формулу Джеффриса; она используется также для столбчатой диаграммы ошибок.

Дополнительные возможности

В синтаксисе вместо фактических значений чисел можно использовать переменные, значения которых являются значениями числителя и знаменателя. Эти переменные могут быть в активном наборе данных или в другом наборе данных, например, сгенерированным функцией AGGREGATE. Затем для каждого наблюдения создается доверительный интервал. Эта процедура не выполняет агрегацию данных.

Это диалоговое окно формирует синтаксис для команды расширения PROPOR. Чтобы вывести справку для этой команды, введите следующую команду:

PROPOR /HELP.
