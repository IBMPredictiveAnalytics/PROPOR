Intervalles de confiance de proportions

Cette procédure génère une table d'intervalles de confiance binomiaux et de Poisson et les différences entre les valeurs.

• Les entrées de cette boîte de dialogue ne proviennent pas d'un jeu de données.

Nombre(s) de numérateurs et Nombre(s) de dénominateurs : Les données de ce calcul sont le numérateur et le dénominateur d'une proportion. Avec un nombre par zone, une table d'une ligne est générée. Avec plusieurs nombres dans chacune, une table contient une ligne par zone et compare chaque intervalle de confiance à la première.

• En bref, si un seul nombre est entré dans une zone, mais que l'autre zone en contient plusieurs, le nombre seul est répliqué autant de fois que nécessaire.

Niveau de confiance : Entrez le niveau de confiance (alpha) à utiliser pour les intervalles sous forme d'un nombre compris entre 0 et 1.

Les intervalles de confiance binomiaux utilisent la formule de Jeffreys qui est également utilisée pour les graphiques de barre d'erreur.

Fonctions supplémentaires

Dans la syntaxe, les variables dont les valeurs correspondent aux nombres de numérateurs et de dénominateurs peuvent être utilisées pour les spécifications de numérateur et de dénominateur au lieu des valeurs réelles des nombres. Les variables peuvent se trouver dans le jeu de données actif ou provenir d'un autre jeu de données (par exemple, généré par AGGREGATE). Un intervalle de confiance est alors généré pour chaque observation. Aucune agrégation de données n'est effectuée par cette procédure.

Cette boîte de dialogue génère une syntaxe pour la commande d'extension PROPOR. Pour afficher l'aide de cette commande, exécutez la syntaxe suivante :

PROPOR /HELP.

Conditions requises

Cette boîte de dialogue requiert le plug-in d'intégration pour Python et la commande d'extension PROPOR. Pour plus d'informations, reportez-vous à Comment obtenir des plug-in d'intégration dans Système central > Questions fréquemment posées dans le système d'aide d'IBM SPSS Statistics.
