Konfidenzintervalle der Anteile

Diese Prozedur erzeugt eine Tabelle mit Binomial- und Poisson-Konfidenzintervallen und Differenzen zwischen den Werten.

• Die Eingaben für diesen Dialog stammen nicht aus einem Dataset.

Zählerwert(e) und Nennerwert(e) Die Daten für diese Berechnung sind der Zähler und der Nenner für einen Anteil. Mit einer Zahl in jedem Feld wird eine einzeilige Tabelle erstellt. Bei mehreren Zahlen in jedem Feld enthält die Tabelle je eine Zeile und bietet einen Vergleich jedes Konfidenzintervalls mit dem ersten Wert.

• Wenn in einem Feld nur eine Zahl eingegeben wird, das andere Feld aber mehr als eine hat, wird die einzelne Zahl so oft wie nötig wiederholt.

Konfidenzniveau Geben Sie das Konfidenzniveau (Alpha), das für die Intervalle verwendet werden soll, als Zahl im Bereich 0 - 1 ein.

Die binomialen Konfidenzintervalle verwenden die Jeffreys-Formel, die auch für Fehlerbalkendiagramme verwendet wird.

Weitere Funktionen

In der Syntax können Variablen, deren Werte die Zähler- und Nennerwerte sind, für die Zähler- und Nennerspezifikationen anstelle der tatsächlichen Zahlenwerte verwendet werden. Die Variablen können im aktiven Dataset oder in einem anderen Dataset enthalten sein, das z. B. von AGGREGATE erzeugt wird. Ein Konfidenzintervall wird dann für jeden Fall erzeugt. Von dieser Prozedur wird keine Datenaggregation ausgeführt.

In diesem Dialogfeld wird Syntax für den Erweiterungsbefehl PROPOR generiert. Führen Sie folgende Syntax aus, um Hilfe für diesen Befehl anzuzeigen:

PROPOR /HELP.
