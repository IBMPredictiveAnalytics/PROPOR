Intervalos de confiança de proporção

Este procedimento produz uma tabela de intervalos de confiança Binomial e Poisson e diferenças entre os valores.

• As entradas para essa caixa de diálogo não vêm de um conjunto de dados.

Contagem(ns) do numerador e contagens do denominador Os dados para esse cálculo são o numerador e o denominador de uma proporção. Com uma contagem em cada campo, uma tabela de uma linha é produzida. Com vários números em cada, a tabela tem uma linha para cada e fornece uma comparação de cada IC com o primeiro.

• Em resumo, se apenas uma contagem for inserida em um campo, mas o outro campo tiver mais de uma, o único número será replicado quantas vezes forem necessárias.

Nível de confiança Insira o nível de confiança (alfa) a ser usado para os intervalos como um número entre 0 e 1.

Os ICs binomiais usam a fórmula de Jeffrey, que também é usada para gráficos de barras de erros.

Recursos adicionais

Na sintaxe, as variáveis cujos valores são as contagens do numerador e do denominador podem ser usadas para as especificações do numerador e do denominador em vez dos valores reais da contagem. As variáveis podem estar no conjunto de dados ativo ou em outro conjunto de dados, como aquele que pode ser produzido por AGGREGATE. Um intervalo de confiança é então produzido para cada caso. Nenhuma agregação de dados é executada por esse procedimento.

Essa caixa de diálogo gera a sintaxe para o comando de extensão PROPOR. Para exibir ajuda para este comando, execute a seguinte sintaxe:

PROPOR /HELP.

Requisitos

Essa caixa de diálogo requer o plug-in de integração para Python e o comando de extensão PROPOR. Para obter informações adicionais, consulte Como Obter Integration Plug-In, no Sistema Principal > Perguntas mais Frequentes no sistema de ajuda do IBM SPSS Statistics Help.
