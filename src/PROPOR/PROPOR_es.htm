Intervalos de confianza de proporción

Este procedimiento produce una tabla de intervalos de confianza Binomiales y Poisson y diferencias entre valores.

• Las entradas para este diálogo no proceden de un conjunto de datos.

Recuentos de numerador y recuentos de denominador Los datos para este cálculo son el numerador y el denominador de una proporción. Con un recuento en cada campo, se produce una tabla de una sola fila. Con varios números en cada uno, la tabla tiene una fila para cada uno y proporciona una comparación de cada CI y con el primero.

• Para abreviar, si solo se introduce un recuento en un campo pero el otro campo tiene más de un, el número individual se réplica tantas veces como sea necesario.

Nivel de confianza Introduzca el nivel de confianza (alfa) que se debe utilizar para los intervalos como un número entre 0 y 1.

Los CI binomiales utilizan la fórmula de Jeffreys, que también se utiliza para los gráficos de barras de errores.

Características adicionales

En la sintaxis, las variables cuyos valores son los recuentos de numerador y de denominador se pueden utilizar para las especificaciones de numerador y denominador en lugar de valores de recuentos reales. Las variables pueden estar en el conjunto de datos activo o ser de otro conjunto de datos, como uno que puede ser producido por AGGREGATE. Luego se produce un intervalo de confianza para cada caso. Este procedimiento no lleva a cabo ninguna agregación de datos.

Este diálogo genera sintaxis para el comando de extensión PROPOR. Para mostrar la ayuda de este comando, ejecute la siguiente sintaxis:

PROPOR /HELP.

Requisitos

Este diálogo requiere el complemento de integración para Python y el comando de extensión PROPOR. Si desea más información, consulte Cómo obtener complementos de integración, en Sistema principal > Preguntas más frecuentes en el sistema de ayuda de IBM SPSS Statistics.
