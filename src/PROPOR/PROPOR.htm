Proportion Confidence Intervals

This procedure produces a table of Binomial and Poisson confidence intervals and differences between values.

• The inputs for this dialog do not come from a dataset.

Numerator Count(s) and Denominator Counts The data for this calculation are the numerator and denominator for a proportion. With one count in each field, a one row table is produced. With multiple numbers in each, the table has one row for each and provides a comparison of each CI with the first.

• As a shorthand, if only one count is entered in a field but the other field has more than one, the single number is replicated as many times as needed.

Confidence Level Enter the confidence level (alpha) to be used for the intervals as a number between 0 and 1.

The binomial CIs use the Jeffreys formula that is also used for error bar charts.

Additional Features

In syntax, variables whose values are the numerator and denominator counts can be used for the numerator and denominator specifications instead of actual count values. The variables can be in the active dataset or from another dataset such as one that might be produced by AGGREGATE. A confidence interval is then produced for each case. No data aggregation is performed by this procedure.

This dialog generates syntax for the PROPOR extension command. To display help for this command, run the following syntax:

PROPOR /HELP.

Requirements

This dialog requires the Integration Plug-In for Python and the PROPOR extension command. For more information, see How to Get Integration Plug-Ins, under Core System > Frequently Asked Questions in the IBM SPSS Statistics Help system.
